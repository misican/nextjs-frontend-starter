# Roles and RACI

## Purpose
Clarify accountability for creating, approving, and maintaining Agile artifacts.

## Core Roles
- Product Owner (PO)
- Frontend Tech Lead (TL)
- QA Lead (QA)
- Engineering Manager / Scrum Master (SM)
- DevOps Lead (DO)
- UX/UI Lead (UX)

## RACI Matrix
| Artifact Group | PO | TL | QA | SM | DO | UX |
|---|---|---|---|---|---|---|
| Concept artifacts | A | R | C | C | I | C |
| Inception artifacts | C | A/R | C | I | C | C |
| Iteration artifacts | A | R | R | R | I | C |
| Release artifacts | A | R | A/R | C | R | C |
| Maintenance artifacts | C | R | A/R | C | R | I |
| Retirement artifacts | A | R | C | C | R | I |
| Sprint packs | A | R | R | A/R | C | C |

## Legend
- **R** Responsible
- **A** Accountable
- **C** Consulted
- **I** Informed
